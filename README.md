<strong>About this mock repository.</strong>
<br>
MockingGitRepo creates a new Git repository from a real source repository while avoiding publication of private implementation details. It replays the source history on the <code>main</code> or <code>master</code> branch, preserves commit messages, commit authors, commit dates, and the file tree shape, then replaces normal source file contents with commit messages written as comments.
<br>
<br>
<strong>Preserved.</strong>
<br>
Commit messages, commit authors and emails, author dates, committer dates, file paths, directory paths, deletions, renames, moves, and README/LICENSE content when source-file preservation is enabled.
<br>
<br>
<strong>Replaced.</strong>
<br>
Normal source files contain mock content. Instead of the original implementation, each touched file accumulates relevant commit messages line by line, formatted with the file type's comment syntax where possible.
<br>
<br>
<strong>Limitations.</strong>
<br>
This repository is not a complete copy of the original project. It is intended to mirror selected Git history for contribution visibility and structural context. Generated content may not build, run, or represent the original implementation.
